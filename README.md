# PropView-Backend

## How to Start/Restart/Stop Server

### In Localhost
> npm run dev

### In Development Server
> npm run dev-start
 
> npm run dev-restart

> npm run dev-stop

### In Production Server
> npm run prod-start

> npm run prod-restart

> npm run prod-stop
